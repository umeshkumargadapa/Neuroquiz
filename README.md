## 📖 Overview
NeuroQuiz is an AI-powered quiz application built with Flask, SQLite and the Groq API. Users can generate quizzes, receive instant feedback, and track quiz history.

## ✨ Features
- Secure Authentication
- AI Quiz Generation
- Quiz History
- Leaderboard
- Responsive UI
- Glassmorphism Design

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Backend | Flask |
| Frontend | HTML, CSS, JavaScript |
| Styling | Tailwind CSS |
| Database | SQLite |
| AI | Groq API |
| Hosting | Render |

## 🚀 Installation

```bash
git clone https://github.com/umeshkumargadapa/NeuroQuiz.git
cd NeuroQuiz
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt
```

Create `.env`

```env
GROQ_API_KEY=your_api_key
```

Run

```bash
python app.py
```
